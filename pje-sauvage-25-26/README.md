# PJE B -- Lucas SAUVAGE 

## Utilisation 

Au préalable, il faut installer un environnement virtuel, ainsi que les dépendences nécessaires. 
Tout ceci est automatisé, il ne vous reste plus qu'à lancer la commande : 

`make install` 

Afin de lancer l'application, merci de lancer 

`make run` 

Si toutefois le Makefile venait à ne pas fonctionner voici les commandes : 

```
$ python3 -m venv venv 
$ pip install -r requirements.txt
$ python3 src/main.py 
```
